React router之exact

Posted on 2018-06-10 |

exact

  • exact是Route下的一条属性，一般而言，react路由会匹配所有匹配到的路由组价，exact能够使得路由的匹配更严格一些。
  • exact的值为bool型，为true是表示严格匹配，为false时为正常匹配。
  • 如在exact为true时，’/link’与’/’是不匹配的，但是在false的情况下它们又是匹配的
